# TypicalAgronomy

Phiên bản được test: 1.21.11 (Paper)

## Những thứ trong plugin

- **Cây trồng tùy chỉnh** — Trồng hạt giống lên farmland, thu hoạch khi chín để nhận item.
- **Agronomy Station** — Máy craft riêng với lưới 3×3, hỗ trợ cả shaped lẫn shapeless recipe.
- **Recipe Browser** — Xem công thức ngay trong game qua cuốn sách trong Agronomy Station.
- **Item Menu** — Dùng lệnh để mở menu duyệt và lấy item theo danh mục.

## Lệnh

| Lệnh | Mô tả |
|---|---|
| `/typicalagronomy items` | Mở menu item |
| `/typicalagronomy reload` | Reload config và lang.yml |
| `/typicalagronomy getitem <ID>` | Lấy item theo ID (admin) |

**Craft Agronomy Station:** 1 Crafting Table + 1 Wheat Seeds (shapeless, bàn craft thường)

## Yêu cầu

- Paper 1.21+
- Java 21+
